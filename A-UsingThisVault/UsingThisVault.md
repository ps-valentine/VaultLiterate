## Welcome!
This Vault is being setup as a digital library for community use. While I haven't built it out for this, look back to this page for more information in the future!

For all listings, I'm playing around with a robust tagging system. For non-fiction books, I'm currently including large excerpts. For fiction books: character lists & plot summaries. Expect arc-level spoilers in summaries for comics. For scientific articles, there will be synopses or quoted abstracts, where I can make them succinctly. Otherwise, I will extract important excerpts of the full text.

If you'd like to branch this project, use [the Github page](https://github.com/ps-valentine/VaultLiterate) to do so. If you'd like to contribute, follow Github process, detailed [here](https://docs.github.com/en/get-started/start-your-journey/hello-world). Your final step will be opening a pull request -- essentially asking me to review your changes. If all is well, I'll incorporate those changes in the vault and it will be a collaborative effort. I am *likely* to edit your additions, as I'm particular about formatting and organization. If you'd like to be credited for you work, include that information in your pull request.

## Documentation Standards
### Plot Summaries
I'll try to keep plot summaries of fiction books folded, using the syntax \"> [!hint]- Plot Summary..." or \">[!hint]- ¡SPOILERS!: Plot Summary...", or something similar. I'll keep them as concise as I can, while giving enough information to use this list to identify recommendations. Avoid reviewing the tags if you want to entirely avoid spoilers, though I try not to go into excessive detail.
### Titling
#### Comics
For comic titling (under Files), the title will be "(Year of Series Start)-Title" or "(Year of Series Start)-AbbreviatedTitle" where necessary.
